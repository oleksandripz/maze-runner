# Maze Runner

## Опис проєкту
Гра "Maze Runner", побудована з використанням React, TypeScript та Vite. 

## Запуск та конфігурація
1. Встановіть залежності: `npm install`
2. Запустіть сервер для розробки: `npm run dev`

### Додаткові команди
- `npm run build` — збірка проєкту для продакшену
- `npm run storybook` — запуск середовища розробки UI компонентів (Storybook)
- `npm run docs` — генерація документації коду
